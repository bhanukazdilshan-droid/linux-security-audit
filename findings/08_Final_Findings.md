# Final Findings

## Project Summary

This project involved performing a complete security audit and hardening assessment on an Ubuntu Server virtual machine.

The assessment included system information gathering, user auditing, network analysis, service review, SSH security verification, firewall configuration, and implementation of security hardening measures.

---

## Security Improvements Implemented

- Reviewed system configuration
- Audited user accounts and privileges
- Verified active network services
- Reviewed SSH configuration
- Investigated authentication logs
- Disabled unnecessary services (CUPS)
- Enabled UFW firewall
- Allowed only SSH access
- Created Bash automation scripts
- Documented all findings with evidence
