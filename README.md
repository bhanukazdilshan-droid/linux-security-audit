# Ubuntu Server Security Audit & Hardening Assessment

## Overview

This project documents a complete security audit and hardening assessment performed on an Ubuntu Server virtual machine.

The objective was to identify security risks, analyze the current system configuration, implement security improvements, and document every step professionally.

This project was completed as part of my cybersecurity portfolio while studying Linux administration, system hardening, Bash scripting, networking, and security monitoring.

---

## Objectives

* Audit Ubuntu Server security
* Analyze users and permissions
* Review network configuration
* Inspect running services
* Monitor system activity
* Configure firewall protection
* Investigate SSH authentication logs
* Create Bash automation scripts
* Produce a professional security report
